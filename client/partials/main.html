  • Show app

    Roomies: Web and iOS.

    Roomies is an app that uses iOS and Angular as simultaeneous front ends, sharing a NodeJS/ExpressJS backend. The app allows users in groups, particularly roommates, to designate tasks and completion times for other users in the group, streamlining the process of assigning responsbilities to individuals. The app utilizes Socket.io to seemlessly update information for all users. It also features as chat system for users in the group.

  • Show app

    Angular Drum Sequencer.

    A javascript drum sequencer built primarily with Angular, run on a node server. The goal of this project was to create an intuitive, visually appealing web sequencer. The web audio library HowlerJs, along with Angular's front end controllers allow easy generation of sounds that can be manipulated by users. Everything is tied together with visual cues made with MaterializeCSS.

  • Show app

    Foodies: Targeted Recipe Builder.

    Foodies is an app built in Python using an MVC called Pylot that allows database interaction with mySQL. The app uses the Food2Fork recipe API. Foodies allows users to create a profile where they can pick food preferences. It then shows data related to those preferences in a constructive way.
